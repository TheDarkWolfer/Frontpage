### Frontpage

A small frontpage to replace all the regular google, qwant and whatever you use as your frontpage !
You're free to use it if you want, just replace the "welcome page" settings in your browser. Also, please
feel free to make your own, so you can have a personalised page to your taste !

Also, it's Tor Friendly ! If you're using the "regular" network, your searches will be done through
DuckDuckGo, but if you're using Tor, it'll instead use the Tor version of DuckDuckGo (duckduckgo.onion) !
That way, you can have a safe *and* gay page !
